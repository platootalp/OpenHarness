欢迎来到 OpenHarness 文档

OpenHarness 是一个开源的 Python Agent Harness，为 LLM 提供工具调用、技能加载、记忆系统和多代理协调功能。

主要特性

  • 43+ 内置工具（文件 I/O、Shell、搜索、Web、MCP）
  • 技能系统（兼容 Claude Code 格式）
  • 持久化记忆系统
  • 多代理协调
  • 权限管理系统

快速开始

# 安装 OpenHarness
pip install openharness

# 运行 Agent
openharness agent

访问 用户指南 了解更多使用方法。
