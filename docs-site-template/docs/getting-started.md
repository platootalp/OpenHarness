# Getting Started

## Prerequisites

- Node.js >= 22.12.0
- npm

## Setup

1. Clone this repository
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:4321

## Writing Docs

Add markdown files to the `docs/` directory. The sidebar is auto-generated from the directory structure.
